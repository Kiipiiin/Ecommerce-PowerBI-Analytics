# Sales & Customer Analytics Dashboard (Power BI)

This project showcases an **interactive multi-page Power BI dashboard** designed to analyze sales performance, customer behavior, and product insights.  
The dashboard focuses on *clarity, storytelling, and actionable insights* through clean visuals and dynamic tooltips.

---

## Features

### Sales Analytics
- Monthly revenue and order trends  
- Channel, country, and vendor performance  
- Product-level revenue comparison  
- Price & quantity contribution breakdown (via tooltip pages)

### Customer Analytics
- New and loyal customer trends  
- Customer segments & acquisition channels  
- Payment method distribution  
- RFM (Recency–Frequency–Monetary) scatter analysis  
- Segment-based deep-dive via tooltip visuals

### Dashboard Design & UX
- Clean multi-page navigation  
- Storytelling-focused titles and subtitles  
- Custom tooltip pages for deeper context  
- Minimalist layout prioritizing clarity and readability

---

## Tools & Technologies
- **Power BI Desktop**  
- DAX (measures for revenue, frequency, RFM, etc.)  
- Data modeling & relationship optimization  
- Custom tooltip report pages
